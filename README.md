# 台中旅遊景點 API

進入畫面後有【搜尋區】，【景點區】。
使用者可以搜尋想要的地區，頁面則會顯示相對應的旅遊景點，可以點進去查看更多的資訊。

## 相關技術

1. 使用 JSON
2. 使用瀑布流呈現畫面
3. 使用 infinite scroll
